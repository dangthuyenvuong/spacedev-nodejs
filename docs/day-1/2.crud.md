# Giới thiệu express

Express là một package quản lý router http hiệu quả và đơn giản

Có các method cơ bản sau:

POST: Thêm dữ liệu (Create)

GET: Lấy dữ liệu (Read)

PUT/PATCH: Update dữ liệu (Update)

DELETE: Xóa dữ liệu (Delete)
